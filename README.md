# Image Gallery.

This is an android app that can show the images using flickr API.

Dependancies:
* Retrofit2
* RxJava
* RxAndroid
* Gson
* Picasso
* Joda-time
* Butterknife
* Dagger2
* Mockito


### Setup
To use this repository, fork/clone it, or download a zip using the green "Clone or download" button at the top of the file list.
